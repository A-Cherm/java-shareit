# Share It

## Описание

Проект является бекэндом сервиса, позволяющего пользователям
делиться вещами, брать их в аренду и оставлять запросы на нужные вещи.

Проект состоит из двух сервисов: основного и шлюза.
Шлюз проводит валидацию данных входящих запросов и перенаправляет их 
в основной сервис, в котором осуществляется вся бизнес логика.

## Технологии и версии

- Java 21
- Spring Boot 3.3.2
- Maven
- Hibernate
- PostgreSQL
- Springdoc-openapi
- Docker

## Установка

Для запуска требуется docker.

Клонируйте проект в [path].

Выполните команду Maven

> mvn install

Выполните команды в терминале

> cd [path]
>
> docker compose up -d

Проект запускается в трёх docker-контейнерах: два сервиса и база данных.

## Документация

Подробное описание эндпойнтов доступно в OpenApi
[документации](https://editor-next.swagger.io/?raw=https://github.com/A-Cherm/java-shareit/tree/main/gateway/openapi.yaml).

Краткое описание функций сервиса:

- создание, получение, обновление и удаление пользователей
- создание, получение и обновление вещей пользователей
- поиск вещей по тексту
- создание, получение и обновление бронирований на вещи пользователей
- создание комментариев к вещам после их аренды
- создание и получение запросов на интересующие вещи.
